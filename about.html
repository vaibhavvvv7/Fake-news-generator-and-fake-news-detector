About the Project

The technology behind the intelligence

The Mission

Credible News is an AI-powered platform designed to demonstrate the dual capabilities of modern foundational models: the ability to generate highly realistic synthetic text, and the ability to detect patterns indicative of artificially fabricated or deceptive content.

By combining generative and discriminative AI into a single interface, this tool serves as both a creative exploration space and a powerful verification utility against the spread of misinformation.

AI Models

GPT-2

Powering the Fiction Generator. An autoregressive language model created by OpenAI that predicts the next word in a sequence to generate coherent human-like text.

BERT Base Uncased

Powering the Credibility Detector. A transformer-based model developed by Google, fine-tuned for sequence classification to predict whether text is real or fake.

Technology Stack

Python

The core programming language driving the machine learning inference and backend logic.

FastAPI

A modern, highly performant web framework for building the APIs that bridge the frontend to the AI models.

PyTorch & Transformers

Hugging Face's Transformers library and PyTorch provide the deep learning infrastructure required to run the heavy AI models locally.

Vanilla Web (HTML/CSS/JS)

The frontend is built without heavy frameworks to ensure maximum speed, utilizing modern CSS features like glassmorphism and variables.
